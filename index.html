MergeVQ: A Unified Framework for Visual Generation and Representation with Token Merging and Quantization

1Zhejiang University 2Tsinghua University 3Westlake University 4The Hong Kong University of Science and Technology 5OPPO AI Center 6CAIR, HKISI-CAS 7Institute of Automation, CASIA 8University of Chinese Academy of Sciences

Abstract

Masked Image Modeling (MIM) with Vector Quantization (VQ) has achieved great success in self-supervised pretraining and image generation. However, most existing methods struggle to address two technical challenges simultaneously. We propose MergeVQ, a unified framework integrating token merging and VQ techniques. It achieves significant performance gains in both visual generation and representation learning while accelerating processes.

Introduction

Vector Quantization (VQ) has been widely used in visual tasks, but existing methods face challenges. We introduce MergeVQ to address these issues. Representation learning and generation are complementary, and MergeVQ decouples and recovers semantics during encoding and reconstruction.

Our contributions include decoupling semantic tokens and source matrix, addressing VQ training challenges, and achieving state-of-the-art performance in both tasks.

Learning Paradigm

3.1 MergeVQ Framework

We introduce the MergeVQ framework with token merge encoding, quantization, token recovery, and reconstruction.

Token Merge Encoding: Given an image \(X \in\)\(\mathbb{R}^{H ×W ×3}\), we employ a two - stage downsampling encoder \(\varepsilon\). First, a CNN layer \(\varepsilon_{1}\) extracts features, producing a feature map \(Z \in \mathbb{R}^{\frac{H}{f} ×\frac{W}{f} ×d}\), where \(f\) is the downsampling factor and \(d\) denotes the channel number. The feature \(Z\) is then flattened into \(L\) - length token sequence \(Z_{L} \in \mathbb{R}^{L ×d}\) as \(Z_{L}=\mathcal{E}_{1}(X)\). Then we employ attention with the merging operation, denoted as \(\varepsilon_{2}\), for second - stage extraction. During this process, we obtain a shorter sequence \(Z_{K} \in \mathbb{R}^{K ×d}\) along with its source matrix \(S \in \mathbb{R}^{K ×L}\) that preserves the spatial relationships of the sequence. This process is expressed as \(Z_{K}, S=\mathcal{E}_{2}\left(Z_{L}\right)\). As such, the entire encoding process can be represented as \(Z_{K}, S=\mathcal{E}(X)\). To ensure that \(Z_{K}\) possess rich semantics, we concurrently impose global alignment on \(Z_{K}\).

Quantization: We employ \(LFQ\) as MergeVQ’s quantization module to minimize the loss of details. Specifically, the codebook is reduced to an integer set and could be denoted as: \(C=\times_{i = 1}^{N}{-1,1}\) \(|C| = 2^{N}\). Thus, the quantization can be summarized as follows: \(z_{K i}=sign(z_{K i})=-1 \cdot \mathbb{I}(z_{K i}<0)+\mathbb{I}(z_{K i}>0)\),where \(z_{K i}\) denotes \(i\) th vector in \(K\) semantic tokens \(Z_{K}\). Then the index of the quantized feature \(z_{m i}\) is formulated as Index \((z_{K i})=\sum_{j = 1}^{N} 2^{k - 1}\cdot\mathbb{I}(z_{K i j}>0)\). Finally, we obtain the quantized semantic tokens, denoted as \(Z_{K q}\): \(Z_{K q}=\mathcal{Q}\left(Z_{K}, \mathcal{C}\right)\).

Token Recovery and Reconstruction: we first perform token - level reconstruction with the recovery module \(R(\cdot, \cdot)\) and source matrix \(S\), which yields a new \(L\) - length \(\hat{Z}_{L}\) as \(\hat{Z}_{L}=\mathcal{R}\left(Z_{q K}, S\right)\). This sequence is then fed into the decoder \(D\) for pixel - level reconstruction, which could be described as \(\hat{X}=\mathcal{D}\left(\hat{Z}_{L}\right)\).

3.2 Harmonize Reconstruction and Representation

Inspired by Masked Image Modeling in representation learning, we use Token Merge and Source Recovery to integrate representation learning. We also impose global alignment constraints.

Efficient Generation

4.1 MergeAR with KV Cache Compression

We introduce MergeAR for efficient autoregressive generation. It leverages token sparsity and a position - recording system to accelerate generation. During training, we sample a merge ratio \(r\), introduce a Merge Instruction Token \(M\), and construct a causal mask. In inference, we use a KV cache to prune repeated tokens.

4.2 Randomized Auto - regressive with Source Recovery

MergeVQ can be implemented by the RandAR generative framework. The \(K\) quantized tokens and source matrix are used for training and generation, and we use the source recovery model and decoder to recover tokens.

Experiments

5.1 Implementation Details

We offer three MergeVQ versions: MergeVQ (G) for pure generation, MergeVQ (G+R) for both generation and representation, and MergeVQ (R) for representation learning only. The encoders have different architectures and parameters. We use AdamW optimizer and various loss functions for training. The visual generator is based on LlaMA - based architecture and is also trained with specific settings.

Conclusion

In this paper, we propose MergeVQ, a unified framework that integrates token merging and quantization techniques for visual generation and representation learning. Our experiments on ImageNet demonstrate that MergeVQ can achieve significant performance gains in both self - supervised pre - training and image generation tasks, while also accelerating the processes. Future work may explore more advanced token merging strategies and further optimize the quantization module to enhance the performance of the framework.
